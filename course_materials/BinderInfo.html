Binder - An online Jupyter Notebook Alternative

Hannah Hodge Waller

17/02/2021

Binder makes a Jupyter Notebook like interface available online.

For best results it should be used with Chrome. It is not compatible with Microsoft Edge.

This is provided as an alternative for users who do not have Jupyter Notebooks or a different Python IDE installed on their computer. It is also useful for dealing with package or file errors (e.g unable to load in Excel files)

Note - This binder repro contains NO SENSITIVE information - and so falls within acceptable use within Government. Please do not upload your own files to Binder. Binder is not a substitute for installing Python on your system, but a stop gap solution for delivering training.

The Introduction to Python Materials can be found at the following link:

https://mybinder.org/v2/gh/hannahhodgewaller/introduction_to_python/HEAD

A screen like this should appear.


Binder may take a while to load; but when loaded a view like a traditional Jupyter Notebook with folders should appear.


To access the course material click Notebooks and then on the relevant chapter.

All packages used in the course are pre installed for you; and are running the latest versions.

All data needed for the course is pre-loaded; and the relative links provided e.g "../data/titanic.csv" works as detailed in the course.

This tool is FREE and as a result it times out after around 15 mins of inactivity e.g not running code.

It will pop up with a message to tell you that it has timed out:


When it does this save your current state by click the cloud symbol with a downwards arrow.


Then reconnect (you may need to refresh the page or click the link again) and use the upwards arrow to restore any code thats missing.

You may then need to choose Run All Above from the cell menu.

Its important that you comment out (with a #) any code cells that do not work, or change them to a markdown cell. This is because Run All stops working when it encounters a cell with an error in it.

The download button allows you to save the file at the end of the session. The current state appears to last for a few days - however we would highly recommend downloading the files.
